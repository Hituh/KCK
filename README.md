# Tetris in terminal

**Controls**
- Move     - ← ↓ →
- Drop     - Space
- Rotate   - ↑
- Pause    - p
- Quit     - ESC
- New Game - Enter

**Points**
- **5** for each new block
- **100** for each burnt line
- best score saved in file *best_score*

**Levels**
- each 10 burnt lines speed will be increased
